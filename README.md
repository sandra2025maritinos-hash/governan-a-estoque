Governança de Dados aplicada ao Pequeno Varejo
### Projeto: Controle de Estoque e Prevenção de Perdas
Este projeto demonstra a aplicação de princípios de **Governança de Dados** e **Business Intelligence** em um cenário real de pequeno comércio, focado na mitigação de riscos operacionais e financeiros.
---
## O Problema (A Brecha)
Identifiquei que a ausência de padrões na coleta de dados de validade e quantidade gerava:
* **Desperdício Financeiro:** Produtos vencendo na prateleira por falta de visibilidade.
* **Ruptura de Estoque:** Perda de vendas por falta de itens críticos.
* **Inconsistência:** Falta de regras de preenchimento, gerando dados "sujos".

## A Solução (Governança na Prática)
Desenvolvi uma ferramenta mobile-first (Google Sheets) com foco em:
1. **Qualidade do Dado:** Implementação de menus suspensos (Data Validation) para evitar erros de digitação.
2. **Integridade:** Bloqueio de fórmulas para evitar erros acidentais via smartphone.
3. **Automação:** Alertas visuais (vermelho/amarelo) para ações proativas de compra e descarte.

## Dashboard de Decisão
Criei um painel visual que traduz o estoque em indicadores estratégicos, permitindo que o gestor foque no que realmente importa.
